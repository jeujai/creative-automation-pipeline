# Product Overview

Creative Automation Pipeline is a command-line tool that automates the generation of social media advertising creative assets using GenAI. The system processes campaign briefs, manages existing assets, generates new images when needed, and produces outputs in multiple aspect ratios (1:1, 9:16, 16:9) with text overlays.

## Key Capabilities

- Process campaign briefs in JSON/YAML format with multiple products
- Reuse existing product assets when available to minimize generation costs
- Generate new hero images via GenAI (OpenAI DALL-E 3) when assets are missing
- Create aspect ratio variants optimized for different social media platforms
- Apply campaign message text overlays with proper contrast and readability
- Organize outputs systematically by campaign, product, and aspect ratio
- Optional brand compliance and legal content checking
- Comprehensive logging and execution reporting

## Target Users

- Campaign managers who need to rapidly generate ad assets
- Creative teams scaling campaigns across products and regions
- Marketing operations automating creative production workflows
